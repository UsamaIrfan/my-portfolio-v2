SWIVR

By Usama Irfan

An online saloon booking mobile app.

This app is built with react native and is supported on both Android and IOS. My role on this application was fixing many bugs on IOS version and some on Android as well like notifications not working, libraries not working properly, replacing incompatible libraries and cleaning the code as well.

The technologies used in this project include

  • React Native
  • Redux
  • React Navigation
  • Skeleton Screens
  • Framer Motion (For slick animations.)
  • React-native-reanimated
  • Lottie React-native
  • React-native-socketio
